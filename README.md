# Inversor Monofásico SPWM ESP32

Proyecto desarrollado en ESP32 utilizando FreeRTOS.

## Módulos

- main.c
- bluetooth.c
- bluetooth.h
- spwm.c
- spwm.h
- buttons.c
- buttons.h

## Funciones

- Generación SPWM
- Control por Bluetooth Classic SPP
- Control mediante botones físicos
- Ajuste de frecuencia de salida
